Основні компоненти:
1. База даних облікових записів: Зберігає інформацію про користувачів, їхні  облікові записи та рівні доступу.
2. Менеджер доступу: Контролює доступ до різних рівнів інформації в залежності від рівня доступу користувача.
3. Інтерфейс користувача: Дозволяє користувачам взаємодіяти з системою (наприклад, реєстрація, авторизація, перегляд даних).

 Опис модулів:

2.1. База даних облікових записів:
Структура користувача: Містить інформацію, таку як ім'я користувача, пароль, рівень доступу.
Функції управління обліковими записами: Додавання, видалення, редагування облікових записів.

2.2. Менеджер доступу:
Аутентифікація: Перевірка імені користувача та пароля.
Авторизація: Перевірка прав доступу користувача для різних рівнів даних.
Шифрування: Забезпечення безпеки даних шляхом шифрування.

2.3. Інтерфейс користувача:
Форма входу: Для аутентифікації користувачів.
Меню користувача: Відображення різних опцій в залежності від рівня доступу.
Функціонал перегляду/редагування даних: Залежно від рівня доступу користувача.

3. Додатковий функціонал:
Логування: Запис дій користувачів для аудиту та моніторингу безпеки.
Відновлення паролів: Механізми для відновлення забутих паролів.
Множинні рівні доступу: Різні рівні доступу для користувачів (наприклад, адміністратор, звичайний користувач).
